# Diba Movie cli



## about :
- You can check out the Diablo Movie site with this script
- You can view the serials and movies trailer with this script. You can also download videos.

### installing :
- `git clone https://github.com/stinerd/Diba_movie.git`
- `chmod +x install.sh`
- `sudo su`
- `./install.sh`
- `chmod +x my_movie.py` 
- `./my_movie.py`
